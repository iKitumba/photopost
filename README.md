# Photopost

This project is a social media sharing image app.

## Dependencies

This project has been made with:

- Node.js;
- ReactJS;
- SQL like database;

## Features

- [x] Authentication
- [x] Register of users
- [x] Generation of JWT
- [x] Avatar uploads with preview
- [x] Image download with preview
- [x] Feed
- [x] Profile page
- [x] Comment the posts
- [x] Logout
